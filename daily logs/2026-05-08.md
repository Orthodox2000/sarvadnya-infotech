## Technical Context for Future Sessions (AI Prompt)

- **Palette Context:** The app uses a CSS-variable driven theme (`:root` in `globals.css` and `theme` object in `lib/theme.ts`). Always use `var(--heading-color)`, `var(--primary-btn-color)`, etc., instead of hardcoded hex codes.
- **Component Pattern:** New interactive sections (like Tutorials) follow the "Modal Overlay" pattern for video playback.
- **Layout Note:** `Navbar` and `Productbar` are global in `layout.tsx`.
- **Image Policy:** All external images must be added to `remotePatterns` in the Next.js config before use. Critical images should use the `priority` prop.
